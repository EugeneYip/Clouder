# fBuilder 使用者指南
網址 - [RUN](https://run.ypcloud.com) -> 登入 -> 選擇 "fBuilder" 然後點擊 "go" 以開始

Flow Builder (fBuilder) 是由 YPCloud 開發的低代碼編程環境，參考了 IBM 的 Node-RED

當您啟動一個流程時，消息將沿著節點鏈傳遞，直到達到終點，前一個節點的輸出將成為下一個節點的輸入。
