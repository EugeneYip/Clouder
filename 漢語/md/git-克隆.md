# 在您的 GitLab 計劃的頁面裡

## 步驟1 - Copy 複製
- 選擇「Clone」並複制 URL（使用 HTTPS 克隆）
 
---

# 打開電腦的終端 Terminal

## 步驟2 - Clone 克隆
```
git clone "您的 git 計劃 url"
```

### 使用 Local 的 UI 界面添加您想要的內容

## 步驟3 
```
cd xxx(您的計劃名稱)
```

## 步驟4 - 拉 Pull
```
git pull
```

## 步驟5 - 推 Push
```
git add .
```
```
git commit -m "xxx"
```
```
git push
```
